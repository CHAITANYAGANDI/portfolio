## 🧩 About this website

A hand-built, single-page portfolio — no templates. Highlights:

- ⚡ **Next.js 16 + React 19** with the App Router, server-rendered for speed and SEO.
- 🎨 **Light & dark mode** with a one-click toggle that remembers your preference (no flash on load).
- 📱 **Fully responsive** — adapts cleanly from large desktops down to mobile, with a collapsible mobile nav.
- ✨ **Framer Motion** scroll animations and a custom SVG system-architecture diagram (no images).
- ♿ **Accessibility-minded** — semantic HTML, keyboard focus states, reduced-motion support, and structured data for search engines.

**Tech stack:** Next.js · React · TypeScript · Tailwind CSS v4 · Framer Motion · Drizzle ORM · deployed on Cloudflare (via [vinext](https://github.com/cloudflare/vinext)).

## 💻 Run it locally

**Prerequisite:** Node.js `>=22.13.0`

```bash
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:3000)
npm run build    # production build
npm start        # serve the production build
```

### Useful commands

| Command | Description |
|---|---|
| `npm run dev` | Start local development |
| `npm run build` | Create the production build |
| `npm start` | Serve the production build |
| `npm run lint` | Run ESLint |
| `npm test` | Build and verify the rendered output |
